# bgv-platform

Minimal monorepo scaffold for bgv-platform.

## Structure

- `client/` — React frontend
- `server/` — Node + Express backend

## Quick start

1. Open two terminals.
2. Start the server:

```bash
cd server
npm install
npm start
```

3. Start the client:

```bash
cd client
npm install
npm start
```
